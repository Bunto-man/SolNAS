# SolNAS

This program is the third installment of my file sharing programs. 

In the case that I make this program public, my research project with a university professor fell through, and I'm posting this independently.

## Purpose

The purpose of this program is to act as proprietary NAS software. I don't like the idea of a server or a computer running a whole operating system just for file sharing when the software can be small enough to run on anything.

## Strength

As stated before, this program should be pretty small and easy enough to use, and also capable of running on anything. 

* Can be run on most computers
* Cross-Platform
* Compatible with IOS through website access
* (Hopefully) Secured using API access keys and password protection for endpoints
* Small RAM impact
* Sleek and modern
* Does not require a lot of setup

## Weaknesses

Being proprietary, and since I am not actually a programmer (surprise) there will be a few flaws in the design. 

* Raid
* Probably no compatibility with Docker
* Probably needs more safety
* configuration cannot be edited from the client view, only the host can edit the config (upload speed, max size, etc)
